Torre de Hanoi

A Torre de Hanoi é um quebra-cabeça matemático que consiste em três pinos e um número de discos de tamanhos diferentes. O objetivo é mover todos os discos de um pino para outro, seguindo algumas regras.

Regras do Jogo

  • Você pode mover apenas um disco de cada vez.
  • Um disco maior nunca pode ser colocado em cima de um disco menor.
  • Você pode usar os três pinos como intermediários para mover os discos.

Como Funciona

O jogo começa com todos os discos empilhados em um dos pinos. O objetivo é mover a pilha completa para outro pino, usando o terceiro pino como auxiliar. A solução mínima para o quebra-cabeça é feita com o seguinte algoritmo recursivo:

  1. Mova n-1 discos do pino inicial para o pino auxiliar.
  2. Mova o disco maior (o disco n) para o pino final.
  3. Mova os n-1 discos do pino auxiliar para o pino final.
